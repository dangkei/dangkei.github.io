2021-07-22 devopts docker 0 评论

Docker Compose是一种工具，旨在帮助定义和共享多容器应用程序。使用 Compose，我们可以创建一个 YAML 文件来定义服务，并且可以使用单个命令来启动或拆除所有内容。

使用 Compose的一大优势是您可以在文件中定义应用程序集，将其保存在项目存储库的根目录下（现在受版本控制），并且可以轻松地让其他人为您的项目做出贡献。有人只需要克隆您的存储库并启动撰写应用程序。事实上，你现在可能会在 GitHub/GitLab 上看到很多项目就是这样做的。
