1. download for windows

https://hub.docker.com/

安装步骤省略 按提示一步步安装即可

2. find a image

  • 我们可以在网站上找一些官方的镜像[Official Images]
    例如 我搜索一下nginx 在结果中 我选择类型为Official Images的直接拿来可以使用

  • 也可以在命令行执行

点击进入后页面右侧会有提示该如何在本地下载该镜像

直接拷贝命令后到本地命令行执行

然后打开 docker-desktop 可以看到本地已经有nginx的镜像可以直接使用了。

3. use the image

点击run就直接在本地运行一个redis的docker容器。
或者使用命令行

docker run 参数很多。 有机会需要的时候可以慢慢研究。这里就不做详细介绍了。

  • 查看本地运行的docker 容器
